肆丨控制线程数量

不限制线程数量，会导致电脑卡死。使用threading模块的Semaphore类或BoundedSemaphore类限制线程数量。初始化Semaphore()会给计数器一个值，每调用一次acquire()，计数器减1，每调用一次release()，计数器加1，当计数器为0时，acquire()调用被阻塞。
