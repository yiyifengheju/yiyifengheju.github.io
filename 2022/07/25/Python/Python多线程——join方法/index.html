Python多线程——join()方法

发表于 2022-07-25 更新于 2022-09-25 分类于 Python

在Python多线程中：

  • join()方法的作用是线程同步。
  • 守护线程为守护其他线程而设立。被守护的主线程不存在后，守护线程自然消失。

壹丨多线程默认情况

默认情况下（setDaemon(False)），主线程执行完后立即退出，子线程继续执行直至结束。

叁丨使用join方法设置同步

子线程使用join()方法后，主线程将被阻塞，直至子线程执行结束。将所有子线程放在一个列表依次使用join()方法，保证全部子线程能运行完成，主线程才退出。
