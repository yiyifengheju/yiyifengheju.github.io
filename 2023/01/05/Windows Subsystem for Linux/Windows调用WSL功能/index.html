Windows调用WSL功能

当安装了WSL之后，在Windows下可以调用Linux的一些命令或功能，如git、sass等

当安装了WSL之后，在Windows下可以调用Linux的一些命令或功能，如git、sass等

壹丨调用WSL-git功能

众所周知，Windows使用git需要安装相关软件，这时候就可以调用Linux自带的git功能

第一步，创建git.cmd脚本

新建文件：C:\Users\MasterMao\git\git.cmd

第二步，添加环境变量

此电脑——属性——高级系统设置——高级-环境变量——Path——编辑——新建，将git.cmd文件路径C:\Users\MasterMao\git写入。

第三步，验证

在Windows PowerShell或Pycharm Terminal中查询git版本：

最后，Pycharm调用WSL-git

  1. 可以在终端中直接使用git命令
  2. 设置——版本控制——Git中，设置Git可执行文件路径为C:\Users\MasterMao\git\git.cmd，点击测试即可看到git版本

贰丨调用sass功能

sass用于前端中将.SCSS、.SASS文件编译成.CSS文件

第一步，WSL中安装sass

第二步，创建sass.bat脚本

新建文件：C:\Users\MasterMao\sass\sass.cmd

第三步，添加环境变量

此电脑——属性——高级系统设置——高级-环境变量——Path——编辑——新建，将sass.cmd文件路径C:\Users\MasterMao\sass写入。

第四步，验证

在Windows PowerShell或Pycharm Terminal中查询sass版本：

最后，WebStorm调用WSL-sass

设置——工具——File Watcher中，新建File Watcher，此时程序应该默认找到了sass工具，确定即可
