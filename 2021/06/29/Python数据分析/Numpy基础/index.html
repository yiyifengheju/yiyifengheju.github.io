Numpy基础

发表于 2021-06-29 更新于 2021-07-12 分类于 Python数据分析 阅读次数：

一、ndarray实例

  • np.array()——生成数组

  • .axis——表示数据维度
  • .rank——轴数量
  • .ndim——秩
  • .shape——n行m列
  • .size——元素个数，=n*m
  • .dtype——元素类型

  • .itemsize——每个元素大小，字节为单位

二、ndarry数据类型

  • bool——True/False
  • intc——与C语言int类型一致，int32/int64

  • intp——用于索引的整数，与C中ssize_t一致，int32/int64

  • int8——字节整数，[-128, 127]
  • int16——16位整数，[-32768, 32767]
  • int32——32位整数，[-2^31, 2^31-1]

  • int64——64位整数，[-2^63, 2^63-1]

  • uint8——8位无符号整数，[0, 255]
  • uint16——[0, 65535]
  • uint32——[0, 2^32-1]

  • uint64——[0, 2^64-1]

  • float16——16位半精度浮点，1位符号位，5位指数，10位尾数
  • float32——32位半精度浮点，1位符号位，8位指数，23位尾数

  • float64——64位半精度浮点，1位符号位，11位指数，52位尾数

  • complex64——复数，实部虚部各32位浮点

  • complex128——复数，实部虚部各64位浮点

三、数组创建

1. 继承

  • np.array(list/tumple)

2. 初始化

  • np.arange(n)——0, n-1整数
  • np.ones(shape)
  • np.zeros(shape)
  • np.full(shape, val)
  • np.eye(n)——n*n单位阵
  • np.ones_like(a)——根据a的形状形成数组
  • np.zeros_like(a)
  • np.full_like(a, val)

3. 其他函数

  • np.linspace()——根据起止数据等间隔填充数据
  • np.concatenate()——两个或多个合并

四、维度变换

  • .reshape(shape)——原数组不变
  • .resize(shape)——原数组改变
  • .swapaxes(ax1, ax2)——调换维度
  • .flatten()——数组降为一维，原数组不变

五、类型变换

  • .astype()，会产生新数组
  • .tolist()，转换为列表

六、索引和切片

  • 一维数组切片
  • 多维数组切片

七、函数

  • .mean()——平均
  • np.abs(x) / np.fabs(x)——绝对值
  • np.sqrt(x)——平方根
  • np.square(x)——平方
  • np.log(x) / np.log10(x) / np.log2(x)
  • np.ceil(x) / np.floor(x)——计算各元素ceiling或floor
