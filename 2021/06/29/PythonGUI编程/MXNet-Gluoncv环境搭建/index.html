MXNet+Gluoncv环境搭建

发表于 2021-06-29 更新于 2022-05-28 分类于 MXNet

本文主要介绍MXNet+Gluoncv环境的搭建

Tips：编程环境随版本稍有变化

Linux下

macOS下

Windows下（对CUDA版本要求严格）

安装前须知

  1. CUDA版本一定为：CUDA 10.1

    查看CUDA版本：

  2. cuDNN版本一定对应 CUDA 10.1

  3. 安装MXNet版本一定为 1.5.0`

  4. Anaconda新建环境，选择 Python 3.6

安装步骤

第一步，安装Microsoft Visual Studio

第二步，安装NVIDIA CUDA-Toolkit，选择10.1 版本，环境变量添加C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v10.1

第三步，下载NVIDIA cuDNN，解压到C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\

第四步，Anaconda新建环境，选择Python 3.6

第五步，安装MXNet

安装中可能出现的问题

出现 OSError: [WinError 126] 找不到指定的模块：

安装微软运行库

libmxnet.dll was not found：

创建Python3.6环境，重新pip install mxnet-cu101==1.5.0
