Windows美化

发表于 2021-06-29 更新于 2022-05-31 分类于 Windows 阅读次数：

本文主要介绍我的 Windows 美化技巧

用过一大票壁纸软件，什么漫天飘落的雪花啦、与鼠标交互的花纹啦、Q弹的XX啦……但使用下来发现，全屏工作或者玩游戏的时间比较多，类似的动态效果简直浪费性能。

壹丨启动项管理

  1. 启动项管理、软件管家、驱动管理：360安全卫士

我的做法是，安装360安全卫士，然后设置中取消勾选所有项目，关闭主动防御，关闭桌面助手……

不得不承认，这种捆绑厂的all in one软件在某些功能上确实好用

  1. 弹窗管理、安全防护：火绒安全

在360安全卫士的启动项管理中，将所有与360相关的启动项禁用，开机就只启动火绒

火绒的弹窗拦截真的超级棒！！

贰丨Win10 磁贴

设置——个性化——开始

在“开始”菜单上显示更多磁贴：ON

在“开始”菜单中显示应用列表：OFF

显示最近添加的应用：OFF

显示最常用的应用：OFF

使用全屏“开始”屏幕：ON

在“开始”菜单或……：OFF

将软件应用全部固定到开始菜单，分组

Tips：使用 Tile Genie（Microsoft Store，免费）可以将图片设为磁贴

个人不习惯将文件夹放在磁贴位置，一般放在桌面或者固定在资源管理器的“快速访问”

叁丨任务栏+桌面

设置——个性化——任务栏

任务栏在屏幕上的位置：顶部

合并任务栏按钮：始终合并按钮

选择哪些图标显示在任务栏上：音量ON，其他全部OFF

Tips：使用TranslucentTB（Microsoft Store，免费）可以设置任务栏透明

桌面保留此电脑、回收站和最常用文件夹，定期清理归档。

肆丨待办事项

选择Microsoft To Do（Microsoft Store，免费），设置开机启动。

个人不推荐参考[1]中的TickTick小部件方案，其他应用一顶置，啥也看不见；桌面背景自动切换到暗色/浅色，字就看不清了。
